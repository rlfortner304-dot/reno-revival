# Grant Documents

This folder contains grant-related documents for the Reno Revival Stability Project, operating under the 501(c)(3) of Reno Revival Church.
## Reno Revival Stability Project

The Reno Revival Stability Project is a faith-based reentry and stability program operating under the 501(c)(3) status of Reno Revival Church. The program supports justice-impacted individuals through mentorship, life skills development, employment support, and community connection.

### Grant Documents Included
- One-page grant summary
- Program narrative
- Outcomes and evaluation plan
- Draft program budget
- Sustainability plan

These documents are intended for private foundations, faith-based funders, and community partners supporting reentry, second-chance, and stabilization initiatives.
Contents will include:
- Grant proposals
- Program narratives
- Budgets
- Letters of support
